# MSc-Biology-Annina-van-Roode+Emmie-Zijlemaker
MSc project Annina van Roode: Courtship in Zebra Finches

## How to use:
1. Download R Studio through this link: https://www.rstudio.com/products/rstudio/download/ 
2. Download the R code about the Zebra Finches, which you can get from Katharina Riebel form Leiden University
3. Download the raw data, which you can get from Katharina Riebel
4. Change lines 26 and 31 in the code to your directory
5. Run the code
